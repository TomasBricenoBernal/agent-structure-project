## Protocol Authority

This is the global protocol for this repository. The source of truth is the
module set in `.agents/protocol/`. `CLAUDE.md` imports those modules with
`@import` (Claude Code expands them at load time). `AGENTS.md` is generated by
flattening the same modules into plain text (run
`python planning/scripts/build_agents.py`) so Codex and opencode, which read
`AGENTS.md` literally, receive the full protocol.

`.claude/agents/*.md` are Claude Code subagents and the single source of truth
for each role's behavior, triggers, tools, and model.

If the protocol modules and a subagent file ever diverge:

- Follow the protocol modules for repository-wide rules (workflow, token budget, review governance).
- Follow the subagent file for that role's own behavior and limits.
- After editing any module, regenerate `AGENTS.md` with `python planning/scripts/build_agents.py` so it stays in sync with `CLAUDE.md`.
