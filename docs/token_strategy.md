# Token Strategy

Use the smallest context required for the task.

## Default Rules

- Start from `tasks/active_task.md`.
- Read only the task files needed for the current work.
- Read related feature files only when the task requires them.
- Avoid broad repository exploration by default.

## Escalate Context Only When Needed

- Architecture or major refactors
- Database design or migrations
- Security-sensitive work
- Complex debugging
